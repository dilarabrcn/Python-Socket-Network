Python Socket Programming - Client/Server Communication
Bu proje, Python kullanılarak geliştirilmiş, TCP/IP protokolü üzerinden haberleşen bir istemci-sunucu (Client-Server) mimarisidir.

Soket Programlama: socket kütüphanesi kullanılarak ağ üzerinden veri iletimi.

Özel Protokol: Verilerin düzenli paketlenmesi ve çözülmesi için geliştirilen protocol.py katmanı.

Çoklu Bağlantı: İstemci ve sunucu arasında eş zamanlı mesajlaşma altyapısı.

server.py: Gelen bağlantıları kabul eden ve yöneten ana sunucu kodu.

client.py: Sunucuya bağlanan ve veri gönderen istemci kodu.

protocol.py: İletişim kurallarını belirleyen yardımcı sınıf.
